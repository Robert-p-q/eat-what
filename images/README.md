# 菜品图片采集指南

## 📁 目录结构

```
images/
├── README.md          ← 本文件
├── 兰州牛肉面-尖椒牛肉拌面.jpg
├── 兰州牛肉面-大盘鸡拌面.jpg
├── 金牌拌饭-招牌烤肉拌饭.jpg
├── ...                 ← 其他菜品图片
```

## 📝 命名规则

图片文件名格式：**`{摊位}-{菜品名}.jpg`**

例如：
- `兰州牛肉面-尖椒牛肉拌面.jpg`
- `金牌拌饭-招牌烤肉拌饭.jpg`
- `炒饭专家-广式炒米粉.jpg`

> ⚠️ 注意：如果不同楼层的同名摊位有同名菜品（如"金牌拌饭"的"沙拉烤肉拌饭"），请加上楼层：`金牌拌饭-2楼-沙拉烤肉拌饭.jpg`

## 📥 采集步骤

1. **打开美团 APP**
2. **搜索食堂 + 摊位名**（如一食堂兰州牛肉面）
3. **找到对应菜品**，截图或保存菜品图片
4. **将图片重命名**（按上方命名规则）
5. **放入 `images/` 文件夹**
6. **更新 `data/foods.json`**，将对应菜品的 `image` 字段改为图片文件名

### 示例

```json
{
  "canteen": "一食堂",
  "floor": "2楼",
  "stall": "兰州牛肉面",
  "name": "尖椒牛肉拌面",
  "price": 18,
  "tags": ["面食", "辣"],
  "image": "images/兰州牛肉面-尖椒牛肉拌面.jpg"
}
```

## 💡 小技巧

- **批量操作**：先收集一批图片统一重命名，再一起更新 JSON
- **推荐图片尺寸**：宽度 600-800px，比例 4:3 或 16:9
- **格式**：JPG 或 WebP，文件大小尽量控制在 200KB 以内
- **可从最常吃的菜开始**：先配 10-20 道常点的，后面慢慢补充

## 📋 待配图菜品清单

运行下面命令可以查看所有待配图菜品：

```bash
# 用 Python 快速检查
python -c "import json; data=json.load(open('../data/foods.json','r',encoding='utf-8')); missing=[d for d in data if not d.get('image')]; print(f'待配图: {len(missing)} 道'); [print(f'  {d[\"canteen\"]} {d[\"stall\"]} - {d[\"name\"]}') for d in missing]"
```
